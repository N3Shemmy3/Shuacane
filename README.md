# Shuacane
A clean weather app built using Nuxt.js & TailwindCSS + Weatherapi.com

## Setup

Make sure to install dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```
